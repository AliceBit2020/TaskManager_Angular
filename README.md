# TaskManager_Angular

Task Manager App is a full-stack web application built with Angular for the frontend and ASP.NET Core Web API for the backend. The project allows users to create, update, delete, and manage tasks with real-time updates and persistent storage using a database.

It demonstrates a modern client-server architecture with REST API integration, clean UI design, and scalable project structure.

🔧 Tech Stack:
Frontend: Angular
Backend: ASP.NET Core Web API
Database: SQL Server
HTTP communication: REST API
